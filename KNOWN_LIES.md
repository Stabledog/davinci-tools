# KNOWN_LIES.md — DaVinci Resolve

This file tracks **commonly repeated but incorrect or obsolete advice** encountered when using DaVinci Resolve.

Entries here override internet tutorials and LLM suggestions unless proven otherwise.

---

## Format

Each entry should include:
- Claim
- Reality
- Affected Versions
- Evidence / Observation
- Status

---

## Example

### Claim
Workspace → Video Clean Feed enables full-screen output on a second monitor.

### Reality
Menu does not exist in Resolve 18.x.

### Affected Versions
Resolve ≥18.0 (Windows)

### Evidence
User observation; menu absent in all Workspace menus.

### Status
Confirmed false
